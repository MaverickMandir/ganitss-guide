# Number Construction

Constructing numbers from digits.

## Question type 1

What is the difference between the largest and the smallest number formed by digits 6,2,7,9and 4 (without repeating any digit)?

> Largest number that could made with this digits is 97642

> Smallest number that could made with this digits is 24679

> Their difference is = 97642 - 24679 = {97642 - 24679}
