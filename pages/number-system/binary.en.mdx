# Binary Number System

The Number system used in computers which only uses 0 and 1.

| Binary Digit | Equivalent Decimal Values |
|:--:|:--:|
| 0 | 0 |
| 1 | 1 |

<Callout type="info" >
  **Binary Number** uses $2^x$ as base value instead of $10^x$ like we use in decimal number. 
</Callout>
